# DRN Satış Yazılımı v1.0

Modern ve esnek bir satış ve mağaza yönetim sistemi.

## Özellikler
- **POS Satış Ekranı:** Hızlı ve kolay satış yapın (barkodlu/barkodsuz).
- **Stok Takibi:** Çoklu depo desteği ile stoklarınızı anlık izleyin.
- **Cari Hesap Yönetimi:** Müşteri borç ve alacak takibi yapın.
- **Gider Takibi:** İşletme masraflarını kaydedin ve yönetin.
- **Etiket Basımı:** Özel şablonlarla hızlı raf etiketi basın.
- **Sadakat Programı:** Müşterilerinize puan kazandırın.

## Kurulum
1. `npm install`
2. `npm run dev`

İyi çalışmalar!
